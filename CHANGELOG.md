# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-19

### Added
- Initial release with `canvas-apps-ui-gen` skill
- Multi-agent YAML generation (layout, controls, styling, assembly specialists)
- 11 reusable YAML templates (cards, forms, tables, sidebar, tab bar, KPI cards)
- Comprehensive reference documentation for PA YAML rules, controls, sizing, and limitations
- Support for three generation modes: replicate from screenshot, improve existing screen, build from text description
